Статья: <https://www.alphaxiv.org/abs/2507.07451>

 Код: <https://github.com/Kwai-Klear/RLEP/tree/main>


Авторы придумали, как можно добавить replay buffer из классического рля.

На первом этапе собирают некоторый пул корректных траекторий с рля у модели. 

На втором этапе начинают новое обучение со стартовой политики и для каждой группы подмешивается рандомно одинаковое количество успешных траекторий из пула
